Using Generative Text AIs to Create Roleplaying Characters & NPCs

Aug 6, 2024 · AI / Unity / Narrative Design

If you've used ChatGPT, you've probably already realized how useful generative text AI can be for writing characters, dialogue, and interactive systems.

Using the API

A basic OpenAI API request is fairly simple. The more interesting part is how you structure the prompt, maintain conversation history, and update the character state.

You can reference variable names inline, like conversationHistories, state, or userInput. 

public class GPT_NPC
{
    public string backgroundInfo = "Here, write the background about your context/world.";

    Dictionary<string, string> _conversationHistories =
        new Dictionary<string, string>
    {
        { "Eldrin", "" },
        { "Aria", "" },
    };

    async void AskChatGPT(string userInput, string character)
    {
        string inputText =
            $"### Background Information:\n" +
            backgroundInfo +
            $"### New Conversation:\n" +
            $"Human: {userInput}\n" +
            $"{character}:";
    }
}

Notice that the code block preserves indentation and uses a different visual container, which makes it much easier to scan.
